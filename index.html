SKILLS

HTML

HTML é uma linguagem de marcação essencial para a construção de páginas na Web. Com ela, é possível estruturar e organizar o conteúdo de um site de forma clara e semântica,

CSS

CSS é uma linguagem utilizada para estilizar documentos HTML. Com o CSS, é possível definir as cores, fontes, espaçamentos e posicionamentos dos elementos de uma página web.

JAVASCRIPT

JavaScript é uma linguagem de programação de alto nível originalmente criada para manipular comportamentos de páginas web e executada principalmente em navegadores.

REACT

React é uma biblioteca JavaScript de código aberto utilizada para criar interfaces de usuário interativas e eficientes. Com o React, é possível construir componentes reutilizáveis que permitem criar interfaces de usuário complexas de maneira modular e escalável.

VUE.JS

Vue é um framework JavaScript progressivo utilizado para a construção de interfaces de usuário interativas e dinâmicas. Com o Vue, é possível criar aplicativos web eficientes e de alto desempenho, com uma sintaxe limpa e simples.

GIT

Git é um sistema de controle de versão distribuído amplamente utilizado para o rastreamento de alterações em projetos de desenvolvimento de software. Com o Git, é possível gerenciar eficientemente as alterações no código-fonte e colaborar com outros desenvolvedores de forma eficaz.

GITHUB

Git é um sistema de controle de versão distribuído amplamente utilizado para o rastreamento de alterações em projetos de desenvolvimento de software. Com o Git, é possível gerenciar eficientemente as alterações no código-fonte e colaborar com outros desenvolvedores de forma eficaz.

PYTHON

Python é uma linguagem de programação versátil e de alto nível, conhecida por sua simplicidade e legibilidade de código. É amplamente utilizada em diversas áreas, como desenvolvimento web, análise de dados e inteligência artificial.
